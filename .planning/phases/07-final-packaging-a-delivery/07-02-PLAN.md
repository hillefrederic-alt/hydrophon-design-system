           name: 'preset-default',

   - Process all brands: hydrophon, gluy, hyhero, hyindustry
   - Process all variants per brand from existing assets/logo/{brand}/svg/

   ```json

     "build:assets": "node scripts/export-assets.js"
   }
   ```
