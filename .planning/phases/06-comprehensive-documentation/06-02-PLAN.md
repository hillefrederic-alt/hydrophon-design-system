# Fuer Entwickler

Technische Spezifikationen, Design Tokens und CSS-Variablen.

## Quick Start

### Design Tokens einbinden
```css
/* CSS-Variablen importieren */
@import 'design-system/build/css/variables.css';

/* Verwendung */
.button-primary {
  background-color: var(--button-primary-background-default);
  color: var(--button-primary-foreground-default);
}
```

### Token-Dateien
| Format | Pfad | Verwendung |
|--------|------|------------|
| JSON (Quelle) | `design-system/tokens/*.json` | Token-Definitionen |
| CSS Variables | `design-system/build/css/variables.css` | Web-Projekte |

## Technische Referenz

-> [Design Tokens Uebersicht](../04-technical-reference/design-tokens.md)
-> [CSS-Variablen Referenz](../04-technical-reference/css-variables.md)
-> [Grid & Breakpoints](../04-technical-reference/grid-breakpoints.md)

## Accessibility Implementation

-> [WCAG 2.1 AA Compliance](../03-accessibility/wcag-compliance.md)

### Kritische ARIA-Patterns
| Komponente | Pattern |
|------------|---------|
| Modal | `role="dialog"`, `aria-modal="true"`, Focus Trap |
| Toast | `role="status"` (success/info), `role="alert"` (warning/error) |
| Breadcrumb | `aria-current="page"` auf aktuellem Element |
| Mobile Drawer | `aria-expanded`, `aria-controls`, ESC zum Schliessen |
