**Structure:**

## 1. Hydrophon Hauptlogo

### Logo-Varianten
- **Original** (Blau + Grau): Standard use on white/light backgrounds
- **Weiß**: For dark backgrounds or colored backgrounds
- **Schwarz**: For single-color printing
- **Blau** (einfarbig): When only brand blue is available
- **Grau** (einfarbig): Subtle/muted applications

Include visual examples with usage context for each.

### Schutzraum (Clearspace)
- Define clearspace as "X" unit (height of the "H" in Hydrophon)
- Minimum clearspace: 1X on all sides
- Visual diagram showing clearspace boundaries
- Extended clearspace recommendations for high-visibility placements

### Mindestgrößen
- **Digital**: Minimum width 120px (ensures legibility)
- **Print**: Minimum width 30mm
- Favicon sizes: 16x16, 32x32, 48x48 (symbol only)
- When width is too small, use symbol-only version if available

### Platzierung
- Preferred positions (top-left for web header, centered for hero)
- Alignment rules with other elements
- Spacing from page edges

### Hintergründe
- Approved background colors (white, light gray, Hydrophon Blau)
- Background contrast requirements
- Photo/image background rules (avoid busy backgrounds)

### Fehlverwendung (Don'ts)
Document at least 8 misuse examples:
1. Do not stretch or distort
2. Do not rotate
3. Do not apply drop shadows
4. Do not place on low-contrast backgrounds
5. Do not crop the logo
6. Do not change colors
7. Do not add effects (gradients, outlines)
8. Do not place too close to other elements

## 2. Produktlinien-Logos

### Gluy
- Variants available: original, weiss, schwarz, hellblau, dunkelblau
- Primary color context: Gelb (#FFEEB6), Hellblau (#88A9C3)
- Same clearspace and size rules as Hauptlogo

### hyHero
- Variants available: original, weiss, schwarz

- Same clearspace and size rules as Hauptlogo

### hyIndustry
- Variants available: original, weiss, schwarz

### Co-Branding
- When Hydrophon logo appears with product line logo
- Hierarchy rules (Hydrophon as parent brand)
- Spacing between logos

## 3. Dateiformate

- **PNG @2x/@3x**: Raster web use, retina displays
- **PDF**: Print production (separate assets folder)
