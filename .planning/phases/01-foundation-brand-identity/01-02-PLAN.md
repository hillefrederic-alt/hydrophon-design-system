<task type="auto">
  <name>Task 1: Create typography token definitions</name>
  <files>design-system/tokens/typography.json</files>
  <action>
Create typography tokens in W3C DTCG format compatible with Style Dictionary 4.0.

**Font Family Recommendations:**
For modern B2B feel with good web availability:
- Headlines: "Inter" (modern, professional) or "DM Sans" (friendly modern)
- Body: "Inter" (excellent readability) or "Source Sans Pro" (professional)
- UI: Same as body for consistency
- Fallbacks: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif

Structure:

1. **Primitive tokens**:
   ```
   fontFamily:
     sans: "Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif"
     mono: "'JetBrains Mono', 'Fira Code', monospace"

   fontSize:
     xs: 12px
     sm: 14px
     base: 16px
     lg: 18px
     xl: 20px
     2xl: 24px
     3xl: 30px
     4xl: 36px
     5xl: 48px
     6xl: 60px

   fontWeight:
     regular: 400
     medium: 500
     semibold: 600
     bold: 700

   lineHeight:
     tight: 1.25
     snug: 1.375
     normal: 1.5
     relaxed: 1.625
     loose: 2

   letterSpacing:
     tighter: -0.05em
     tight: -0.025em
     normal: 0
     wide: 0.025em
     wider: 0.05em
   ```

2. **Semantic tokens**:
   ```
   typography:
     heading:
       h1: { fontSize: 5xl, fontWeight: bold, lineHeight: tight, letterSpacing: tight }
       h2: { fontSize: 4xl, fontWeight: bold, lineHeight: tight }
       h3: { fontSize: 3xl, fontWeight: semibold, lineHeight: snug }
       h4: { fontSize: 2xl, fontWeight: semibold, lineHeight: snug }
       h5: { fontSize: xl, fontWeight: medium, lineHeight: normal }
       h6: { fontSize: lg, fontWeight: medium, lineHeight: normal }

     body:
       large: { fontSize: lg, lineHeight: relaxed }
       base: { fontSize: base, lineHeight: normal }
       small: { fontSize: sm, lineHeight: normal }

     ui:
       button: { fontSize: sm, fontWeight: medium, letterSpacing: wide }
       label: { fontSize: sm, fontWeight: medium }
       caption: { fontSize: xs, lineHeight: normal }
   ```

Use $value and $type syntax per W3C DTCG spec.
  </action>
  <verify>JSON validates with `npx jsonlint design-system/tokens/typography.json`</verify>
  <done>typography.json contains complete font families, sizes, weights, line heights, and semantic styles</done>
