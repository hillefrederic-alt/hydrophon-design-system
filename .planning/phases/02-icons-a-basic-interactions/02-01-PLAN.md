<task type="auto">
  <name>Task 2: Create icon documentation</name>
  <files>design-system/docs/icons.md</files>
  <action>
Create comprehensive German icon documentation (minimum 150 lines) covering:

**1. Icon-Stil (ICON-01)**
- Visual specifications: 2px stroke weight, rounded caps (border-radius: 2px), rounded joins
- Grid system: 24x24px base grid with 2px padding (20x20 safe area)
- Why these choices: Modern B2B aesthetic, consistency with Inter font curves, industry standard
- Example visual showing stroke weight and grid

**2. Icon-Bibliothek (ICON-02)**
- Primary library: Lucide Icons (ISC license, 2px customizable stroke)
- Installation: npm install lucide-react (or framework-agnostic SVG usage)
- Link to Lucide website: https://lucide.dev
- List of recommended icons by category:

Navigation (8-10 icons):
- menu, x (close), chevron-down, chevron-right, chevron-left, arrow-left, arrow-right, arrow-up, arrow-down, home

Actions (10-12 icons):
- search, filter, download, upload, share, edit, trash-2, plus, minus, copy, check, external-link

Status/Feedback (6-8 icons):
- check-circle, alert-circle, info, alert-triangle, loader, x-circle, bell

Product-specific (placeholder guidance):
- Note that custom product icons (drain systems, plumbing fixtures, certifications) should follow same 2px stroke, 24x24 grid
- Guidance for creating custom icons matching Lucide style

**3. Größen und Anwendung (ICON-03)**
Table showing when to use each size:
| Größe | Pixel | Anwendung | Beispiel |
|-------|-------|-----------|----------|
| XS | 16px | Inline mit kleinem Text, kompakte UIs | Breadcrumb-Chevrons |
| SM | 20px | Buttons mit Text, Formulare | Button "Speichern" |
| MD | 24px | Standalone-Icons, Icon-Buttons | Toolbar-Aktionen |
| LG | 32px | Hervorgehobene Aktionen | Feature-Highlights |
| XL | 48px | Dekorative Elemente | Leere-State-Illustrationen |

**4. Barrierefreiheit (WCAG AA)**
- aria-hidden="true" for decorative icons (accompanying text)
- aria-label on parent button for icon-only buttons
- role="img" + aria-label for informative icons
- Touch target minimum 44x44px for interactive icons
- Code examples for each pattern

**5. Export-Formate (ICON-04)**
- Primary: SVG with currentColor for fill/stroke
- Fallback: PNG at 2x/3x for legacy systems
- SVGO optimization settings for clean exports
- viewBox standardization: "0 0 24 24"

**6. Don'ts**
- Nicht: Icons mit Text versehen (verhindert Lokalisierung)
- Nicht: Willkürliche Größen verwenden (17px, 22px)
- Nicht: Farbwerte hardcoden (immer currentColor)
- Nicht: Fokusindikatoren bei interaktiven Icons weglassen
