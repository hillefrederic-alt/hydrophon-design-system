# Requirements: Hydrophon Design-System

**Defined:** 2026-01-28
**Core Value:** Das Design-System vermittelt Hydrophons moderne, innovative Positionierung im B2B-Sanitärbereich und ermöglicht konsistente Markenkommunikation über alle Touchpoints hinweg.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Interaktive Design-System-Website | Statische Dokumentation reicht für v1, Web-Tool kann später kommen |
| Framework-spezifische Komponenten (React/Vue) | Website-Framework noch nicht entschieden, Code-agnostisch |
| Automatische Code-Generation | Overengineering für v1, manueller Export reicht |
| Print-spezifische Guidelines | Fokus auf Digital, Print später bei Bedarf |
| Detaillierte Copy-/Tone-of-Voice-Guidelines | Visuelles System hat Priorität |
| Produktlinien-spezifische Sub-Brands | Alle nutzen Hydrophon-System, keine separaten Identitäten |
| AR/VR Guidelines | Zu früh, 3D-Standards erst in v2 |
| Contribution Model | Wird erst bei größerer Adoption relevant |
